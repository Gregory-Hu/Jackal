# Job Description: 高级 Python 开发工程师

## 职位概述

高级 Python 开发工程师是技术团队的核心成员，负责后端服务的设计、开发和优化。

## 核心职责

### 1. 开发工作
- 设计和实现高质量的 Python 代码
- 开发 RESTful API 和微服务
- 优化现有系统性能

### 2. 代码审查
- 审查团队成员的代码
- 确保代码符合规范
- 提供建设性反馈

### 3. 架构设计
- 参与系统架构设计
- 评估技术方案
- 编写设计文档

### 4. 团队协作
- 指导初级工程师
- 与产品经理沟通需求
- 与测试团队协作

## 技能要求

### 必需技能
| 技能 | 熟练度 |
|------|--------|
| Python 编程 | 专家 |
| Web 框架 (Flask/Django/FastAPI) | 熟练 |
| 数据库设计 | 熟练 |
| API 设计 | 熟练 |

### 优先技能
- 系统设计
- 代码审查
- 团队指导

## 工作约束

1. **权限限制**
   - 不能直接修改生产环境
   - 数据库变更需要审批

2. **流程要求**
   - 代码必须经过审查
   - 重大变更需要 RFC 文档

## 成功指标

- 代码质量指标
- 项目交付及时率
- 团队协作评价
