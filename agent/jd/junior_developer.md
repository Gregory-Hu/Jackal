# Job Description: 初级开发工程师

## 职位概述

初级开发工程师在团队中学习和成长，完成基础开发任务。

## 核心职责

### 1. 开发任务
- 实现功能模块
- 修复分配的 Bug
- 编写单元测试

### 2. 学习成长
- 学习团队技术规范
- 阅读和理解现有代码
- 参与技术分享

### 3. 文档工作
- 编写功能文档
- 更新 API 文档
- 记录学习笔记

## 技能要求

### 必需技能
| 技能 | 熟练度 |
|------|--------|
| Python 基础 | 熟练 |
| Git 使用 | 基础 |
| 单元测试 | 基础 |

### 优先技能
- Web 开发基础
- 数据库基础

## 工作约束

1. **代码审查**
   - 所有代码必须经过审查
   - 根据反馈及时修改

2. **权限限制**
   - 无生产环境权限
   - 数据库操作需审批

## 成功指标

- 任务按时完成
- 代码质量提升
- 学习能力体现
